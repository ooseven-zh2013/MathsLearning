13 三角函数基础：用角度描述形状

你已经学过三角形的内角和是 $180^\circ$。现在把"角度"和"边长"联系起来——这就是三角函数。

一、直角三角形回顾

任意一个直角三角形：


       C（直角）
       │\
       │ \    斜边（最长边）
 对边 │  \   ↖
       │   \  ↙
       A ───┘ B
        └───────┘
          邻边
  • 直角在 $C$ 点（$\angle C = 90^\circ$）
  • 斜边：直角所对的边（最长）
  • 对边：相对于某个锐角的对边
  • 邻边：相对于某个锐角的邻边（不是斜边）

二、勾股定理（你已经听说过）

📖 勾股定理：直角三角形两直角边的平方和 $=$ 斜边的平方。

$$a^2 + b^2 = c^2$$

其中 $c$ 是斜边，$a$ 和 $b$ 是两直角边。

例子：直角边为 $3$ 和 $4$ 的直角三角形，斜边 $= \sqrt{3^2+4^2} = \sqrt{25} = 5$。

三、三角函数的定义

在直角三角形 $ABC$ 中（$\angle C = 90^\circ$），设锐角为 $\alpha$（在 $A$ 点）：

正弦 $\sin \alpha$

$$\sin \alpha = \frac{\text{对边}}{\text{斜边}}$$

余弦 $\cos \alpha$

$$\cos \alpha = \frac{\text{邻边}}{\text{斜边}}$$

正切 $\tan \alpha$

$$\tan \alpha = \frac{\text{对边}}{\text{邻边}}$$

🔑 记忆口诀：SohCahToa

  • Sin = Opposite / Hypotenuse（对边/斜边）
  • Cos = Adjacent / Hypotenuse（邻边/斜边）
  • Tan = Opposite / Adjacent（对边/邻边）

四、特殊角的三角函数值

你需要记住这几个特殊角的值：

角度 $\alpha$$\sin \alpha$$\cos \alpha$$\tan \alpha$
角度 $\alpha$$\sin \alpha$$\cos \alpha$$\tan \alpha$
$0^\circ$$0$$1$$0$
$30^\circ$$\frac{1}{2}$$\frac{\sqrt{3}}{2}$$\frac{1}{\sqrt{3}}$
$45^\circ$$\frac{\sqrt{2}}{2}$$\frac{\sqrt{2}}{2}$$1$
$60^\circ$$\frac{\sqrt{3}}{2}$$\frac{1}{2}$$\sqrt{3}$
$90^\circ$$1$$0$不存在

💡 怎么记忆：

  • $\sin$ 值：$0 \to \frac{1}{2} \to \frac{\sqrt{2}}{2} \to \frac{\sqrt{3}}{2} \to 1$（从小到大）
  • $\cos$ 值：和 $\sin$ 值"反着来"（$\cos \alpha = \sin(90^\circ-\alpha)$）

五、基本恒等式

📖 毕达哥拉斯恒等式：

$$\sin^2\alpha + \cos^2\alpha = 1$$

用勾股定理证明：

  • $\sin \alpha = \frac{\text{对边}}{\text{斜边}}$，$\cos \alpha = \frac{\text{邻边}}{\text{斜边}}$
  • $(\text{对边})^2 + (\text{邻边})^2 = (\text{斜边})^2$（勾股定理）
  • 两边除以 $(\text{斜边})^2$：

$\left(\frac{\text{对边}}{\text{斜边}}\right)^2 + \left(\frac{\text{邻边}}{\text{斜边}}\right)^2 = 1$

$\Rightarrow \sin^2\alpha + \cos^2\alpha = 1$ ✓

六、相似三角形：为什么三角函数是"确定的"？

你也许会问："如果三角形大小不同，$\sin \alpha$ 的值还一样吗？"

答案是：一样！

因为相似三角形对应边的比例相同。只要角度 $\alpha$ 相同，不管三角形多大，$\sin \alpha$ 的值都一样。

这就是为什么 $\sin \alpha$ 是一个"函数"：给定 $\alpha$，$\sin \alpha$ 的值唯一确定。

七、三角函数的图像（直观介绍）

如果你学过坐标系，可以把 $\sin$ 和 $\cos$ "画出来"：

$y = \sin x$（周期函数，永远在 $-1$ 和 $1$ 之间震荡） 1 · · │ / │ / │ / 0 ·─────┼─────→ x │ / │ / -1 · $y = \cos x$（和 $\sin$ 很像，只是"平移"了 $90^\circ$） 1 · · │ / │ / │ / 0 ·─────┼─────→ x │ / │ / -1 ·

🔑 核心特性：

  • $\sin$ 和 $\cos$ 的值永远在 $-1$ 和 $1$ 之间
  • 它们是周期函数：每转一圈（$360^\circ$ 或 $2\pi$ 弧度），值重复

八、弧度制（非常重要的概念！）

你已经习惯用"度"（$^\circ$）来量角。但微积分中用的不是度，而是弧度。

📖 弧度：圆心角所对的弧长与半径之比。


    半径 = r
    ╭───────╮
    │   弧长为 r  │
    │            │
    └────────────┘
    这条弧所对的圆心角 = 1 弧度（rad）

度和弧度的换算：

$$180^\circ = \pi \text{ 弧度}$$

所以：

  • $360^\circ = 2\pi$ rad
  • $180^\circ = \pi$ rad
  • $90^\circ = \dfrac{\pi}{2}$ rad
  • $60^\circ = \dfrac{\pi}{3}$ rad
  • $45^\circ = \dfrac{\pi}{4}$ rad
  • $30^\circ = \dfrac{\pi}{6}$ rad

💡 为什么要用弧度？ 因为在微积分中，只有用弧度，才有：

$$\lim_{x \to 0} \frac{\sin x}{x} = 1 \quad (x \text{ 用弧度})$$

如果用度，这个极限就不等于 $1$ 了！

  2. $\sin 30^\circ + \cos 60^\circ = \;?$
  3. 用弧度表示：$90^\circ$、$180^\circ$、$270^\circ$、$360^\circ$。
  4. 为什么 $\sin^2\alpha + \cos^2\alpha = 1$？用你自己的话解释。
  5. $\tan 45^\circ = \;?$
点击查看答案
  1. $\sin^2\alpha + \cos^2\alpha = 1 \rightarrow (\frac{3}{5})^2 + \cos^2\alpha = 1 \rightarrow \cos^2\alpha = 1 - \frac{9}{25} = \frac{16}{25} \rightarrow \cos \alpha = \frac{4}{5}$（正值，因为 $\alpha$ 是锐角）
  1. $\sin 30^\circ = \frac{1}{2}$，$\cos 60^\circ = \frac{1}{2}$ → 和 $= 1$
  1. $90^\circ = \frac{\pi}{2}$ rad，$180^\circ = \pi$ rad，$270^\circ = \frac{3\pi}{2}$ rad，$360^\circ = 2\pi$ rad
  1. 勾股定理 $a^2 + b^2 = c^2$，两边除以 $c^2$：$(\frac{a}{c})^2 + (\frac{b}{c})^2 = 1 \rightarrow \sin^2\alpha + \cos^2\alpha = 1$
  1. $\tan 45^\circ = 1$（对边等于邻边）
