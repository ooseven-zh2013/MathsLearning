16 不定积分：导数的逆运算

你已经会求导了（从函数得到导数）。现在反过来：从导数还原出原来的函数。这叫做"不定积分"。

一、原函数的概念

如果 $F'(x) = f(x)$，那么 $F(x)$ 就是 $f(x)$ 的一个原函数。

换句话说：对 $F$ 求导，得到 $f$。

例子：

$f(x) = 2x$ 问：哪个函数求导之后等于 $2x$？ 试一试：$F(x) = x^2$ $F'(x) = 2x$ ✓ ← 对了！ 所以 $F(x) = x^2$ 是 $f(x) = 2x$ 的一个原函数。

二、一个函数有多少个原函数？

接着上面的例子：

$F(x) = x^2$ → $F'(x) = 2x$ ✓ $G(x) = x^2 + 1$ → $G'(x) = 2x$ ✓ （常数导数为 $0$） $H(x) = x^2 - 5$ → $H'(x) = 2x$ ✓

📖 结论：如果 $F(x)$ 是 $f(x)$ 的一个原函数，那么 $F(x) + C$（$C$ 是任意常数）都是 $f(x)$ 的原函数。

💡 为什么？因为 $(C)' = 0$，加个常数不影响导数。

三、不定积分的符号

📖 不定积分：求 $f(x)$ 的所有原函数的过程，记作：

$$\int f(x) dx = F(x) + C$$

其中：

  • $\int$：积分符号（拉长的大写 S，来自 Sum）
  • $f(x)$：被积函数
  • $dx$：对 $x$ 积分（积分变量）
  • $C$：积分常数（任意常数）

🔑 核心公式：$\displaystyle\int f(x) dx = F(x) + C$，其中 $F'(x) = f(x)$

四、基本积分表（和导数公式一一对应）

导数公式积分公式
导数公式积分公式
$(x)' = 1$$\displaystyle\int 1 \, dx = x + C$
$(x^2)' = 2x$$\displaystyle\int 2x \, dx = x^2 + C$
$(x^3)' = 3x^2$$\displaystyle\int 3x^2 \, dx = x^3 + C$
$(x^n)' = n \cdot x^{n-1}$$\displaystyle\int n \cdot x^{n-1} \, dx = x^n + C$
$(\sin x)' = \cos x$$\displaystyle\int \cos x \, dx = \sin x + C$
$(\cos x)' = -\sin x$$\displaystyle\int (-\sin x) \, dx = \cos x + C$

⚠️ 注意：积分公式可以从导数公式倒推出来！

五、幂函数积分法则

最常用、最重要的积分公式：

🔑 $$\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$

验证（倒推）：

对右边求导： $$\left(\frac{x^{n+1}}{n+1} + C\right)' = \frac{n+1}{n+1} \cdot x^n + 0 = x^n \; \checkmark$$

例子：

$\displaystyle\int x^2 dx = \frac{x^3}{3} + C$ $\displaystyle\int x^7 dx = \frac{x^8}{8} + C$ $\displaystyle\int x^{-2} dx = \frac{x^{-1}}{-1} + C = -\frac{1}{x} + C$ （$n = -2 \neq -1$，合法）

⚠️ $n = -1$ 的情况：$\displaystyle\int \frac{1}{x} dx = \ln|x| + C$（这个初一阶段不要求）

六、积分的线性性质

和求导一样，积分也有线性性质：

🔑 $$\int [k \cdot f(x)] dx = k \cdot \int f(x) dx$$（常数可以提出来）

🔑 $$\int [f(x) \pm g(x)] dx = \int f(x) dx \pm \int g(x) dx$$（和的积分 = 积分的和）

例子：

$$ \begin{aligned} \int (3x^2 + 2x) dx &= 3\int x^2 dx + 2\int x dx \\ &= 3 \cdot \frac{x^3}{3} + 2 \cdot \frac{x^2}{2} + C \\ &= x^3 + x^2 + C \end{aligned} $$

验证（求导）：$(x^3 + x^2 + C)' = 3x^2 + 2x$ ✓

七、用积分求原函数（练习）

例 1：$f(x) = 6x$

$$\int 6x dx = 6 \cdot \int x dx = 6 \cdot \frac{x^2}{2} + C = 3x^2 + C$$

验证：$(3x^2 + C)' = 6x$ ✓

例 2：$f(x) = x^3 + 2x - 5$

$$ \begin{aligned} \int (x^3 + 2x - 5) dx &= \int x^3 dx + 2\int x dx - 5\int 1 dx \\ &= \frac{x^4}{4} + 2 \cdot \frac{x^2}{2} - 5x + C \\ &= \frac{x^4}{4} + x^2 - 5x + C \end{aligned} $$

验证：$(\frac{x^4}{4} + x^2 - 5x + C)' = x^3 + 2x - 5$ ✓

八、积分常数 $C$ 的物理意义

不定积分里的 $C$ 不是摆设——它代表初始条件。

比如：已知加速度 $a(t) = 2$（常数），求速度 $v(t)$。

$$v(t) = \int a(t) dt = \int 2 dt = 2t + C$$

$C$ 是什么？——初始速度（$t = 0$ 时的速度）！

如果已知 $v(0) = 3$，那么 $C = 3$，所以 $v(t) = 2t + 3$。

✅ 检查点

  1. 求 $\displaystyle\int 4x^3 dx$。
  2. 求 $\displaystyle\int (x^2 + 3x) dx$。
  3. 已知 $F'(x) = 2x$ 且 $F(0) = 5$，求 $F(x)$。
  4. 验证：对 $\displaystyle\int (3x^2 + 1) dx$ 的结果求导，看是否回到 $3x^2 + 1$。
点击查看答案
  1. $\displaystyle\int 4x^3 dx = 4 \cdot \int x^3 dx = 4 \cdot \frac{x^4}{4} + C = x^4 + C$。
  1. $\displaystyle\int (x^2 + 3x) dx = \frac{x^3}{3} + 3 \cdot \frac{x^2}{2} + C = \frac{x^3}{3} + \frac{3}{2}x^2 + C$。
  1. $F(x) = \displaystyle\int 2x dx = x^2 + C$。由 $F(0) = 5$ 得：$0 + C = 5 \to C = 5$。所以 $F(x) = x^2 + 5$。
  1. $\displaystyle\int (3x^2 + 1) dx = x^3 + x + C$。求导：$(x^3 + x + C)' = 3x^2 + 1$ ✓
