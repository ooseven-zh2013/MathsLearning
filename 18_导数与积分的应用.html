18 导数与积分的应用：学以致用

学了导数和积分，能干什么？本章把所有工具串起来，解决实际问题。

一、用导数分析函数的性质

1. 单调性（你已经知道直觉）

🔑 用导数判断单调性：

$f'(x)$ 的符号函数行为
$f'(x)$ 的符号函数行为
$f'(x) > 0$函数递增（$x$ 变大，$y$ 也变大）
$f'(x) < 0$函数递减（$x$ 变大，$y$ 变小）
$f'(x) = 0$可能是极值点（最高点或最低点）

例子：分析 $f(x) = x^2 - 4x + 3$


$f'(x) = 2x - 4$

当 $2x - 4 > 0 \to x > 2$：$f'(x) > 0$ → 递增
当 $2x - 4 < 0 \to x < 2$：$f'(x) < 0$ → 递减

所以：

  • 在 $(-\infty, 2)$ 上递减
  • 在 $(2, +\infty)$ 上递增
  • $x = 2$ 是最低点（极小值点）

二、极值点：最高点和最低点

📖 极值：函数在某点附近的最高或最低值。

寻找极值的步骤：

  1. 求导数 $f'(x)$
  2. 解方程 $f'(x) = 0$，找到驻点
  3. 判断驻点左右 $f'(x)$ 的符号变化
符号变化极值类型
符号变化极值类型
$f'(x)$：$+ \to -$极大值（局部最高点）
$f'(x)$：$- \to +$极小值（局部最低点）
$f'(x)$：不变号不是极值（拐点）

例子：$f(x) = x^3 - 3x$

$$f'(x) = 3x^2 - 3 = 3(x^2 - 1) = 3(x-1)(x+1)$$

驻点：$x = 1$ 和 $x = -1$

在 $x = -1$ 附近：

  • $x < -1$：$f'(x) > 0$（递增）
  • $x > -1$（但 $< 1$）：$f'(x) < 0$（递减）
  • $\to$ $+ \to -$ ：极大值点！

在 $x = 1$ 附近：

  • $x < 1$（但 $> -1$）：$f'(x) < 0$（递减）
  • $x > 1$：$f'(x) > 0$（递增）
  • $\to$ $- \to +$ ：极小值点！

三、简单的物理应用

1. 运动学：位置、速度、加速度

回忆一下：

$$s(t) \rightarrow \text{位置函数}$$ $$v(t) = s'(t) \rightarrow \text{速度（位置的导数）}$$ $$a(t) = v'(t) = s''(t) \rightarrow \text{加速度（速度的导数）}$$

反过来（积分）：

$$v(t) = \int a(t) dt + v_0 \quad (v_0 = \text{初速度})$$ $$s(t) = \int v(t) dt + s_0 \quad (s_0 = \text{初始位置})$$

例子：自由落体

物体从静止自由下落（忽略空气阻力）：

  • 加速度 $a(t) = g \approx 9.8$ m/s²（常数）
  • 速度：$v(t) = \int g dt = gt + v_0$，$v_0 = 0 \to \mathbf{v(t) = gt}$
  • 位置：$s(t) = \int gt dt = \frac{1}{2}gt^2 + s_0$，$s_0 = 0 \to \mathbf{s(t) = \frac{1}{2}gt^2}$

四、用积分算面积

1. 曲线下的面积

你已经学过了：$\displaystyle\int_a^b f(x) dx =$ 曲线与 $x$ 轴之间的面积（$f(x) \geq 0$ 时）。

2. 两条曲线之间的面积

如果有两个函数 $f(x)$ 和 $g(x)$，且 $f(x) \geq g(x)$ 在 $[a, b]$ 上：

$$\text{面积} = \int_a^b (f(x) - g(x)) dx$$

    y
    ↑
    │     f(x)
    │   ⋂⋯⋯⋅
    │  /      ← $f(x) - g(x) =$ 两条线之间的距离
    │ /       
    │/ g(x)  
  ──┼───────→ x
    a       b

例子：求 $y = x^2$ 和 $y = x$ 在 $[0, 1]$ 之间的面积

在 $[0,1]$ 上：$x \geq x^2$（直线在上方）

$$ \begin{aligned} \text{面积} &= \int_0^1 (x - x^2) dx \\ &= \left(\frac{x^2}{2} - \frac{x^3}{3}\right)\Big|_0^1 \\ &= \left(\frac{1}{2} - \frac{1}{3}\right) - (0 - 0) \\ &= \frac{1}{6} \; \checkmark \end{aligned} $$

五、最大值和最小值（优化问题入门）

现实问题中经常需要找"最大"或"最小"：

怎样才能让面积最大？ 怎样才能让成本最低？

基本思路：

  1. 建立函数 $f(x)$（面积、成本等）
  2. 求导数 $f'(x)$
  3. 解 $f'(x) = 0$ 找驻点
  4. 比较驻点和端点的函数值 → 最大的就是最大值，最小的就是最小值

例子：用 $100$ 米篱笆围最大面积的矩形

设矩形长 $= x$，宽 $= y$ 周长：$2x + 2y = 100 \to x + y = 50 \to y = 50 - x$

面积：$A(x) = x \cdot y = x(50 - x) = 50x - x^2$

求最大值： $A'(x) = 50 - 2x = 0 \to x = 25$ 此时 $y = 50 - 25 = 25$

→ 正方形时面积最大！最大面积 $= 25 \times 25 = 625$ 平方米 ✓

六、积分在概率中的直观介绍（选学）

你也许听说过"正态分布"。它的曲线下面是钟形曲线：


    y
    ↑
  1 ·     ⋂
    │    / \
    │   /   \
    │  /     \
    │ /       \
  ──┼─────────→ x

曲线下的总面积 $= 1$（概率的总和是 $100\%$）。

这需要用积分来严格定义，但你现在只需要知道：积分 $=$ 累积 $=$ 总面积。

✅ 检查点

  1. $f(x) = x^2 - 4x + 5$ 在 $x = 2$ 附近是递增还是递减？$x = 2$ 是极值点吗？
  2. 自由落体中，如果初速度 $v_0 = 10$ m/s，求 $v(t)$ 和 $s(t)$（$a = g = 9.8$）。
  3. 求 $y = x$ 和 $y = x^2$ 在 $[0, 2]$ 之间围成的面积。
  4. 用导数证明：周长为定值时，矩形中正方形的面积最大。

点击查看答案

  1. $f'(x) = 2x - 4$。在 $x = 2$ 时 $f'(2) = 0$。

$x < 2$：$f'(x) < 0$（递减） $x > 2$：$f'(x) > 0$（递增） → $x = 2$ 是极小值点（最低点）。$f(2) = 4 - 8 + 5 = 1$。

  1. $v(t) = \int g dt + v_0 = gt + 10$

$s(t) = \int (gt+10) dt = \frac{1}{2}gt^2 + 10t + s_0$（$s_0 = 0$ 假设从原点开始） → $s(t) = 4.9t^2 + 10t$

  1. 在 $[0,1]$ 上：$x \geq x^2$；在 $[1,2]$ 上：$x^2 \geq x$

面积 $= \displaystyle\int_0^1 (x - x^2) dx + \displaystyle\int_1^2 (x^2 - x) dx$ $= \left(\frac{1}{2} - \frac{1}{3}\right) + \left(\frac{8}{3} - 2 - \frac{1}{2} + \frac{1}{3}\right)$ $= \frac{1}{6} + \frac{3}{6} = \mathbf{\dfrac{2}{3}}$ ✓

  1. 设长 $x$ 宽 $y$，周长 $2(x+y) = L$（常数）→ $y = \frac{L}{2} - x$

面积 $A(x) = x(\frac{L}{2} - x) = \frac{L}{2}x - x^2$ $A'(x) = \frac{L}{2} - 2x = 0 \to x = \frac{L}{4}$ 此时 $y = \frac{L}{2} - \frac{L}{4} = \frac{L}{4} = x$ → 正方形 ✓
