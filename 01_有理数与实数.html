01 数系的扩充：从有理数到实数

你已经会做有理数的加减乘除了。但世界上不只有"能写成分数的数"。本章带你认识无理数，完成数系的最后一次扩充。

一、回顾：什么是有理数？

📖 有理数：能写成两个整数之比（分母不为零）的数。

例如：

  • $3$（$= \frac{3}{1}$，整数也是有理数）
  • $-5$（$= -\frac{5}{1}$）
  • $\frac{1}{2}$（分数，显然是有理数）
  • $0.75$（$= \frac{3}{4}$，有限小数也是有理数）
  • $0.333...$（$= \frac{1}{3}$，无限循环小数也是有理数！）

🔑 判断标准：有限小数或无限循环小数 = 有理数。

✅ 检查点：$-7$、$0$、$\frac{2}{5}$、$0.666...$、$3.14159$ 中哪些是有理数？

点击查看答案

全都是！$-7 = -\frac{7}{1}$，$0 = \frac{0}{1}$，$\frac{2}{5}$ 是分数，$0.666... = \frac{2}{3}$（循环），$3.14159$ 是有限小数。

二、问题：对角线有多长？

画一个边长为 $1$ 的正方形：


    D ┌───────┐ C
      │       │
      │       │  边长 = 1
      │       │
    A └───────┘ B

对角线 $AC$ 有多长？

根据勾股定理：

在直角三角形 $ABC$ 中：

  • 直角边 $AB = 1$，直角边 $BC = 1$
  • 斜边 $AC^2 = 1^2 + 1^2 = 2$
  • 所以 $AC = \sqrt{2}$

$\sqrt{2}$ 等于多少？

算一算：


1.4² = 1.96   （太小）
1.5² = 2.25   （太大）
1.41² = 1.9881 （小了）
1.42² = 2.0164 （大了）
1.414² = 1.999396
1.415² = 2.002225
...

你会发现：$\sqrt{2}$ 的小数部分永远不会循环，也永远不会终止。

它约等于 $1.41421356237...$，后面有无穷多位，而且没有规律。

三、无理数的诞生

📖 无理数：不能写成两个整数之比（即不能写成分数）的数。

等价地说：无限不循环小数就是无理数。

更多无理数的例子：

数 说明
数 说明
$\sqrt{2} \approx 1.4142...$ 边长1的正方形对角线
$\sqrt{3} \approx 1.7320...$ 常见于等边三角形
$\pi \approx 3.14159265...$ 圆的周长与直径之比
$e \approx 2.71828...$ 自然常数，后面会常见到

⚠️ 常见误区：

  • $3.14$ 是有理数（有限小数），但 $\pi$ 是无理数！
  • $\sqrt{4} = 2$ 是有理数（开方能开尽的不是无理数）
  • $\frac{22}{7}$ 是有理数（它是 $\pi$ 的近似值，但不等于 $\pi$）

三（续）、根式入门

🔑 根式：$\sqrt[n]{a}$ 表示 "$a$ 的 $n$ 次方根"。

  • $\sqrt{a}$ 即 $\sqrt[2]{a}$，平方根。$x^2 = a \Rightarrow x = \pm\sqrt{a}$（$a \ge 0$）
  • $\sqrt[3]{a}$，立方根。$x^3 = a \Rightarrow x = \sqrt[3]{a}$（$a$ 可以是负数）

🔑 基本运算规则（$a \ge 0,\ b \ge 0$）：

  • $\sqrt{a} \times \sqrt{b} = \sqrt{ab}$
  • $\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}$
  • $(\sqrt{a})^2 = a$
  • $\sqrt{a^2} = |a|$（注意：不是 $a$，是 $|a|$！）

🔑 化简根式：把能开出来的因子提到根号外。

化简 √12：
√12 = √(4 × 3) = √4 × √3 = 2√3

化简 √50：
√50 = √(25 × 2) = 5√2

✅ 检查点：化简 $\sqrt{8}$、$\sqrt{18}$、$\sqrt{27}$。

点击查看答案

$\sqrt{8} = \sqrt{4 \times 2} = 2\sqrt{2}$；$\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}$；$\sqrt{27} = \sqrt{9 \times 3} = 3\sqrt{3}$。

四、实数：完整的数轴

📖 实数集 $\mathbb{R}$：所有有理数和所有无理数的集合。


实数 R
├── 有理数 Q
│   ├── 整数 Z（... -2, -1, 0, 1, 2 ...）
│   │   └── 自然数 N（0, 1, 2, 3 ...）
│   └── 分数（1/2, -3/4, 0.75, 0.333...）
└── 无理数（√2, π, e, ...）

🔑 实数轴：每一个实数都对应数轴上的唯一一个点。


    -2    -1     0     1    √2   2     3
─────┼──────┼──────┼──────┼───│──┼──────┼──→
     │      │      │      │   │  │      │

注意：数轴上没有"空隙"。有理数虽然密密麻麻，但它们之间有"洞"——那些洞就是无理数。实数填满了所有洞。

五、实数的基本运算

好消息：实数的加减乘除规则和你学过的有理数一模一样！

  • 加法交换律：$a + b = b + a$
  • 加法结合律：$(a + b) + c = a + (b + c)$
  • 乘法交换律：$a \times b = b \times a$
  • 乘法结合律：$(a \times b) \times c = a \times (b \times c)$
  • 分配律：$a \times (b + c) = a \times b + a \times c$

同样：正数 $> 0$，负数 $< 0$，绝对值表示到原点的距离。

💡 对无理数的运算：实际计算时通常保留根号形式，或者取近似值。

例如：

  • $\sqrt{2} + \sqrt{2} = 2\sqrt{2}$（而不是 $\approx 2.828$）
  • $\sqrt{2} \times \sqrt{2} = 2$
  • $\sqrt{2} \times \sqrt{3} = \sqrt{6}$

六、实数的大小比较

方法：平方比较法（对于正数）。

要比较 $\sqrt{5}$ 和 $\sqrt{7}$：

  • $(\sqrt{5})^2 = 5$，$(\sqrt{7})^2 = 7$
  • 因为 $5 < 7$，所以 $\sqrt{5} < \sqrt{7}$

要比较 $2$ 和 $\sqrt{5}$：

  • $2^2 = 4$，$(\sqrt{5})^2 = 5$
  • 因为 $4 < 5$，所以 $2 < \sqrt{5}$

通法：两边同时平方（确保都是正数），然后比较。

七、为什么需要实数？

你也许觉得有理数就够用了。确实，在日常生活中，我们通常只需要几位小数。但在数学中：

  1. 解方程：$x^2 = 2$ 的解是 $\pm\sqrt{2}$，不是有理数。如果我们只有有理数，这个方程就"无解"。
  2. 连续性：微积分需要"连续"的概念——数轴不能有洞。
  3. 极限：很多极限的结果是无理数（例如 $\pi$）。

🔑 实数 = 有理数 + 无理数 = 完整的数轴 = 微积分的基础。

✅ 检查点

在进入下一章之前，确保你能回答：

  1. $0.101001000100001...$ 是有理数还是无理数？为什么？
  2. 以下哪些是有理数？$3.14$、$\sqrt{9}$、$\sqrt{10}$、$\frac{1}{3}$、$\frac{\pi}{\pi}$
  3. 比较 $\sqrt{8}$ 和 $3$ 的大小。
  4. 实数和有理数的关系，用一个比喻来描述。

点击查看答案

  1. 无理数。它不循环（每次多一个 $0$），也不是有限小数。
  2. $3.14$（有限小数）、$\sqrt{9} = 3$（整数）、$\frac{1}{3}$（分数）、$\frac{\pi}{\pi} = 1$（等于 $1$）都有理。$\sqrt{10}$ 是无理数。
  3. $(\sqrt{8})^2 = 8$，$3^2 = 9$。$8 < 9$，所以 $\sqrt{8} < 3$。
  4. 有理数像数轴上的"钉子"，密密麻麻但仍有空隙；无理数填满了空隙，合在一起就是没有缝隙的实数轴。
