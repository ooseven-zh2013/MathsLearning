07 二次函数：抛物线的世界

一次函数画出来是直线。如果把 $x$ 的指数从 $1$ 变成 $2$，会变出什么花样？答案是——抛物线。

一、最简单的二次函数：$y = x^2$

先看最纯粹的版本：$y = x^2$。

用描点法画出图像：

这条 U 形曲线就是抛物线。

🔑 关键特征：

  • 开口向上
  • 关于 $y$ 轴对称（$(-1)^2 = 1^2 = 1$）
  • 最低点（顶点）在 $(0, 0)$

二、一般形式：$y = ax^2 + bx + c$

📖 二次函数的标准形式：$y = ax^2 + bx + c$（$a \neq 0$）

其中：

  • $a$：决定开口方向和"宽窄"
  • $b$：影响对称轴的位置
  • $c$：图像与 $y$ 轴的交点（截距）

三、系数 $a$ 的作用

$a$ 的正负决定开口方向


$a > 0$：开口向上 ⋃（像个 U）    $a < 0$：开口向下 ⋂（像个拱门）


$a = 2$（开口窄，变化快）    $a = 0.5$（开口宽，变化慢）

     │ y                   │ y
  6 ··     $y=2x^2$       4 ·    $y=0.5x^2$
  4 ··                2 ·  ·
  2 ··                0 ·······
  ──┼──→ x            ──┼──→ x

🔑 $|a|$ 越大，抛物线越"瘦"；$|a|$ 越小，抛物线越"胖"。

四、系数 $c$ 的作用

$c$ 就是图像与 $y$ 轴的交点。因为当 $x = 0$ 时，$y = 0 + 0 + c = c$。


$y = x^2 + 3$：顶点在 $(0, 3)$，图像整体向上平移了 $3$
$y = x^2 - 2$：顶点在 $(0, -2)$，图像整体向下平移了 $2$
$y = x^2$    ：顶点在 $(0, 0)$

五、对称轴和顶点

对于 $y = ax^2 + bx + c$：

🔑 对称轴：图像左右对称的那条竖直线。

对称轴的方程：$x = -\dfrac{b}{2a}$

🔑 顶点：抛物线的最高点（$a < 0$）或最低点（$a > 0$）。

顶点的 $x$ 坐标 $= -\dfrac{b}{2a}$，代入函数得到 $y$ 坐标。

例子：$y = x^2 - 4x + 3$

  • $a = 1$, $b = -4$, $c = 3$
  • 对称轴：$x = -\dfrac{(-4)}{2 \times 1} = 2$
  • 顶点 $x$ 坐标 $= 2$，$y = 2^2 - 4 \times 2 + 3 = 4 - 8 + 3 = -1$
  • 顶点：$(2, -1)$

六、二次函数的三种形式

1. 一般式：$y = ax^2 + bx + c$

最通用的写法，但不容易直接看出图像特征。

2. 顶点式：$y = a(x - h)^2 + k$

顶点直接就是 $(h, k)$！

例如 $y = 2(x - 3)^2 + 1$：顶点 $(3, 1)$，$a = 2 > 0$ 开口向上。

3. 交点式（因式分解形式）：$y = a(x - x_1)(x - x_2)$

$x_1$ 和 $x_2$ 是图像与 $x$ 轴的两个交点（零点/根）。

例如 $y = (x - 1)(x - 3)$：与 $x$ 轴交于 $(1, 0)$ 和 $(3, 0)$。

💡 不同形式在不同问题中各有用处。

七、图像变换：平移

你已经学过平面图形的平移。函数图像也可以平移！

变换公式变化效果

向右平移 $h$$x \to x - h$顶点 $x$ 坐标 $+ h$
向左平移 $h$$x \to x + h$顶点 $x$ 坐标 $- h$

一般规则：

  • $y = a(x - h)^2$ 是 $y = ax^2$ 向右平移 $h$ 个单位
  • $y = a(x - h)^2 + k$ 是 $y = ax^2$ 向右平移 $h$、向上平移 $k$

八、图像与 $x$ 轴的交点（解方程）

二次函数的图像与 $x$ 轴相交时，交点的 $x$ 坐标满足 $y = 0$，即：

$ax^2 + bx + c = 0$

这恰好是一元二次方程。

图像交点个数含义
图像交点个数含义
与 $x$ 轴交于两点$2$方程有两个不同的实数解
与 $x$ 轴相切于一点$1$方程有两个相等的实数解
完全在 $x$ 轴上方或下方$0$方程没有实数解

✅ 检查点

  1. $y = 2x^2$ 和 $y = -x^2$ 的图像各是什么形状的？开口向哪？
  2. 求 $y = x^2 + 6x + 5$ 的顶点坐标和对称轴。
  3. $y = (x - 2)^2 + 3$ 的顶点在哪？
  4. 写出 $y = x^2$ 向左移 $3$ 个单位、向下移 $2$ 个单位后的函数表达式。

点击查看答案

  1. $y = 2x^2$：开口向上的抛物线，比较"瘦"。$y = -x^2$：开口向下的抛物线。
  1. $a = 1$, $b = 6$：对称轴 $x = -\dfrac{b}{2a} = -\dfrac{6}{2} = -3$。顶点：$y = (-3)^2 + 6(-3) + 5 = 9 - 18 + 5 = -4$。顶点 $(-3, -4)$。
  1. 顶点式直接读：顶点 $(2, 3)$。
  1. $y = (x + 3)^2 - 2$（注意：向左移是 $x \to x + 3$，向下移是整体减 $2$）
