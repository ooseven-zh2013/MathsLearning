06 一次函数：最简单的变化规律

你也许见过 $y = 2x + 1$ 这样的式子。这就是"一次函数"——最基础的函数，也是最容易画图、最容易理解的函数。

一、变量和常量

在数学中，我们用字母表示数。这些字母分成两类：

  • 变量：数值会变化的量。通常用 $x$、$y$ 表示。
  • 常量：固定不变的数。通常用 $a$、$b$、$c$、$k$ 表示。

例如：$y = 3x + 5$ 中，$x$ 和 $y$ 是变量（变化），$3$ 和 $5$ 是常量（固定）。

二、函数的概念（初次相遇）

📖 函数：对于变量 $x$ 在某个范围（定义域）内的每一个取值，变量 $y$ 都有唯一确定的值与之对应。

我们记作 $y = f(x)$。

💡 把函数想象成一台机器：


  x → ┌─────────┐ → y
 输入   │  规则 f  │  输出
      └─────────┘

你扔进去一个数 $x$，机器按照某种规则，吐出一个数 $y$。

🔑 关键词："每一个" + "唯一"。

  • "每一个"：定义域中的每个 $x$ 都要有对应的 $y$
  • "唯一"：同一个 $x$ 不能对应两个不同的 $y$

三、一次函数的标准形式

📖 一次函数：形如 $y = kx + b$（$k \neq 0$）的函数。

其中：

  • $k$：斜率（决定直线的倾斜程度和方向）
  • $b$：截距（直线与 $y$ 轴的交点纵坐标）

例子：

函数$k$$b$特点
函数$k$$b$特点
$y = 2x + 1$$2$$1$过 $(0, 1)$，向右上倾斜
$y = -x + 3$$-1$$3$过 $(0, 3)$，向右下倾斜
$y = 0.5x$$0.5$$0$过原点 $(0, 0)$，正比例函数

四、斜率 $k$ 的含义

斜率 $k$ 表示：$x$ 每增加 $1$，$y$ 增加多少。


y = 2x + 1：
  当 x 从 0 变成 1：y 从 1 变成 3（增加了 2）
  当 x 从 1 变成 2：y 从 3 变成 5（增加了 2）
  每次 x 增加 1，y 固定增加 2 ← 这就是 k = 2

图像上，$k$ 决定了直线有多"陡"：


 k = 3： 很陡 ↗        k = 0.5： 平缓 →↗
 k = 1： 45° ↗         k = -1： 45° ↘
 k = 0： 水平 →        k < 0： 向下倾斜 ↘

💡 $k$ 的正负：

  • $k > 0$：$y$ 随 $x$ 增大而增大（递增函数）
  • $k < 0$：$y$ 随 $x$ 增大而减小（递减函数）
  • $k = 0$：$y$ 恒等于 $b$（常函数，一条水平线）

五、截距 $b$ 的含义

$b$ 表示：当 $x = 0$ 时，$y$ 的值。

在图像上，$b$ 就是直线与 $y$ 轴的交点的纵坐标：

$y = 2x + 3$：交 $y$ 轴于 $(0, 3)$ $y = 2x$ ：交 $y$ 轴于 $(0, 0)$，经过原点 $y = 2x - 1$：交 $y$ 轴于 $(0, -1)$

六、画一次函数的图像

只需两点就能确定一条直线！

方法：

  1. 找截距：点 $(0, b)$ — 这是与 $y$ 轴的交点
  2. 利用斜率再找一个点：$x$ 加 $1$，$y$ 加 $k$（即点 $(1, b+k)$）
  3. 或直接算两个点，连起来

例子：画 $y = 3x - 2$

方法一：

  • 截距 $(0, -2)$
  • 斜率 $3$ 意味着：$x$ 加 $1$，$y$ 加 $3$ → 下一个点 $(1, 1)$
  • 连起来

方法二：

  • 当 $x = 0$，$y = -2$ → $(0, -2)$
  • 当 $x = 2$，$y = 4$ → $(2, 4)$
  • 连起来

两种方法得到同一条直线。

七、正比例函数：一次函数的特例

当 $b = 0$ 时，$y = kx$，叫做正比例函数。

特点：图像一定经过原点 $(0, 0)$。

$y = 3x$：过 $(0,0)$ 和 $(1,3)$ $y = -2x$：过 $(0,0)$ 和 $(1,-2)$

本质上，正比例函数就是"$y$ 始终是 $x$ 的固定倍数"。

八、用一次函数解决问题

例子 1：打车计费

起步价 $8$ 元，之后每公里 $2$ 元。

设里程为 $x$（公里），总价为 $y$（元）：

$y = 2x + 8$
  • 斜率 $2$：每多 $1$ 公里多 $2$ 元
  • 截距 $8$：起步价（$0$ 公里也要付 $8$ 元）

例子 2：温度转换

华氏度 $F$ 和摄氏度 $C$ 的关系：

$F = 1.8C + 32$
  • 当 $C = 0$（冰点），$F = 32$
  • 当 $C$ 增加 $10$，$F$ 增加 $18$

✅ 检查点

  1. 对于 $y = -2x + 5$，$k$ 和 $b$ 分别是多少？图像与 $y$ 轴交于哪个点？
  2. 画 $y = x - 3$ 的图像。
  3. 已知直线过 $(0, 2)$ 和 $(3, 8)$，求它的表达式 $y = kx + b$。
  4. 以下哪些函数图像经过原点？$y = 3x$、$y = x + 1$、$y = -0.5x + 0$、$y = 2x - 1$
点击查看答案
  1. $k = -2$，$b = 5$。与 $y$ 轴交于 $(0, 5)$。
  2. 过 $(0, -3)$ 和 $(1, -2)$ 的直线，向右上方倾斜 $45^\circ$。
  3. $b = 2$（截距），$k = \frac{8-2}{3-0} = 2$。所以 $y = 2x + 2$。
  4. $y = 3x$（$b=0$）和 $y = -0.5x + 0$（$b=0$）。实际上 $b=0$ 就经过原点。
