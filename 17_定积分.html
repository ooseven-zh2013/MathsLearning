17 定积分：曲边梯形的面积

用矩形面积的公式，你只能算"直边图形"的面积。但曲线围成的图形呢？这就是定积分要解决的问题。

一、问题：曲线下面积是多少？

看一个具体例子：$y = x^2$ 和 $x$ 轴在 $[0, 1]$ 区间围成的面积。

这个面积不是矩形，也不是三角形——它是曲边梯形。

二、思路：切分成无数个小矩形

核心思想（黎曼的思想）：

  1. 把 $[0, 1]$ 区间切成很多小段
  2. 每一小段上，用矩形来近似曲线下的面积
  3. 把所有矩形面积加起来
  4. 让小矩形的宽度趋近于 $0$，近似就变成精确！

    切 4 段时：         切 8 段时：
    ·                     ·
    │ *                   │ **
    │ **                  │ ****
    │ ****                │ ******
    │ ******             │ ********
    └───────┼───→ x     └───────┼───→ x
    （近似很粗糙）       （近似好一些了）

🔑 关键：当矩形宽度 $\to 0$ 时，矩形面积之和的极限值 $=$ 曲线下面积的精确值。

三、定积分的符号

📖 定积分：函数 $f(x)$ 在区间 $[a, b]$ 上的定积分记作：

$$\int_a^b f(x) dx$$

其中：

  • $a$：积分下限（左端点）
  • $b$：积分上限（右端点）
  • $f(x)$：被积函数
  • $dx$：对 $x$ 积分（$x$ 是积分变量）

四、微积分基本定理（牛顿-莱布尼茨公式）

这是微积分中最重要、最漂亮的定理：

🔑 如果 $F'(x) = f(x)$（$F$ 是 $f$ 的一个原函数），那么：

$$\int_a^b f(x) dx = F(b) - F(a)$$

用符号记：$F(x)\Big|_a^b = F(b) - F(a)$

💡 大白话：定积分 $=$ 原函数在上下限处的差值。

不用再算极限了！找到原函数，代入上下限，减一下，就出来了。

五、用基本定理算第一个定积分

例 1：算 $\displaystyle\int_0^1 x^2 dx$

步骤 1：找 $x^2$ 的一个原函数。

$\displaystyle\int x^2 dx = \frac{x^3}{3} + C$ 取 $F(x) = \dfrac{x^3}{3}$（$C$ 可以任取，一般会消掉）

步骤 2：代入上下限。

$$ \begin{aligned} \int_0^1 x^2 dx &= F(1) - F(0) \\ &= \frac{1^3}{3} - \frac{0^3}{3} \\ &= \frac{1}{3} - 0 \\ &= \frac{1}{3} \; \checkmark \end{aligned} $$

答案：$y = x^2$ 在 $[0,1]$ 下面积 $= \dfrac{1}{3}$。

例 2：算 $\displaystyle\int_0^2 (2x + 1) dx$

步骤 1：找原函数。

$\displaystyle\int (2x+1) dx = 2 \cdot \frac{x^2}{2} + x + C = x^2 + x + C$ 取 $F(x) = x^2 + x$

步骤 2：代入。

$$ \int_0^2 (2x+1) dx = F(2) - F(0) = (4 + 2) - (0 + 0) = 6 \; \checkmark $$

六、定积分的几何意义

$\displaystyle\int_a^b f(x) dx$ 的几何意义：

  • 如果 $f(x) \geq 0$（在 $x$ 轴上方）：$=$ 曲线与 $x$ 轴之间的面积
  • 如果 $f(x) \leq 0$（在 $x$ 轴下方）：$=$ 面积的负值（曲线在下方）
  • 如果 $f(x)$ 有正有负：$=$ 上方面积减去下方面积

    正面积：  ↗           负面积：  ↘
    y ↑                   y ↑
    │  ⋂                 │  ⋃
    │ / \                │   / \
  ──┼────→ x           ──┼────→ x

💡 技巧：要求"总面积"（不管正负），需要对正负部分分段积分，再取绝对值相加。

七、定积分的性质

性质 1：上下限颠倒

$$\int_a^b f(x) dx = -\int_b^a f(x) dx$$

性质 2：区间可加性

$$\int_a^c f(x) dx = \int_a^b f(x) dx + \int_b^c f(x) dx$$

（$a < b < c$）

性质 3：常数倍

$$\int_a^b k \cdot f(x) dx = k \cdot \int_a^b f(x) dx$$

八、用定积分算面积（综合例子）

例：求 $y = x$ 在 $[0, 3]$ 上与 $x$ 轴围成的面积


    y
    ↑
  3 ·           ·
    │          /
    │        /
    │      /
    │    /
    │  /
  0 └───────┼────────→ x
               3

这是个三角形！底 $= 3$，高 $= 3$，面积 $= \frac{3 \times 3}{2} = \frac{9}{2}$。

用定积分验证：

$$ \int_0^3 x dx = \frac{x^2}{2}\Big|_0^3 = \frac{9}{2} - 0 = \frac{9}{2} \; \checkmark $$

两种算法答案一样！

✅ 检查点

  1. 用基本定理算：$\displaystyle\int_1^4 3x^2 dx = ?$
  2. $\displaystyle\int_0^\pi \sin x dx = ?$（提示：$\sin x$ 的原函数是 $-\cos x$）
  3. 判断题：$\displaystyle\int_a^b f(x) dx = F(a) - F(b)$。对还是错？
  4. 用定积分表示"$y = 2x$ 在 $[0, 5]$ 上与 $x$ 轴围成的面积"，并算出结果。
点击查看答案
  1. 原函数：$\displaystyle\int 3x^2 dx = x^3 + C$。$F(4)-F(1) = 64 - 1 = \mathbf{63}$。
  1. $\displaystyle\int \sin x dx = -\cos x + C$。

$(-\cos \pi) - (-\cos 0) = -(-1) - (-1) = 1 + 1 = \mathbf{2}$。

  1. 错！应该是 $F(b) - F(a)$，次序不能反。
  1. $\displaystyle\int_0^5 2x dx = x^2\Big|_0^5 = 25 - 0 = \mathbf{25}$。

（验证：底 $5$ 高 $10$ 的三角形面积 $= \frac{5 \times 10}{2} = 25$ ✓）
