04 集合论基础：万物的容器

"集合"听起来很高级，其实就是一个"框"，把一些东西放进去。用集合的视角，很多数学概念会变得特别清晰。

一、什么是集合？

📖 集合：把一些确定的对象看作一个整体。

关键词是"确定的"——对于任意一个东西，必须能明确判断它"在不在"这个集合里。

例子：

  • 你班上的所有同学 → 一个集合
  • 所有正整数 → 一个集合
  • "比较高的同学" → 不是集合（多高算高？不确定）

💡 元素：集合里的每个东西叫做这个集合的元素。

二、基础符号

属于：$\in$

如果 $a$ 是集合 $A$ 的元素，记作 $a \in A$（读作"$a$ 属于 $A$"）。

不属于：$\notin$

如果 $a$ 不是 $A$ 的元素，记作 $a \notin A$。

空集：$\emptyset$

没有任何元素的集合，记作 $\emptyset$（或 $\{\}$）。

例子：

  • "大于 $5$ 且小于 $4$ 的实数"的集合 → $\emptyset$（没有这样的数）

三、集合的表示法

方法一：列举法

把元素一个一个写在大括号里。

A = {1, 2, 3, 4, 5} ← 1 到 5 的整数 B = {2, 3, 5, 7, 11, $\dots$ } ← 全体质数（用 $\dots$ 表示"以此类推"）

方法二：描述法

用语言描述这个集合的特征。

C = $\{x \mid x \text{ 是大于 } 0 \text{ 的实数}\}$ ← "所有大于 $0$ 的实数"

D = $\{x \mid x^2 = 4\}$ ← "平方等于 $4$ 的数"，即 $\{2, -2\}$

竖线 "$\mid$" 读作 "满足" 或 "使得"。

描述法的一般格式：

$\{元素的一般形式 \mid 元素满足的条件\}$

四、常用数集

数学中有几个数集有专用符号，你需要记住：

符号 名称 含义 例子
$\mathbb{N}$ 自然数集 $\{0, 1, 2, 3, \dots\}$ $3 \in \mathbb{N}$，$-1 \notin \mathbb{N}$
$\mathbb{Z}$ 整数集 $\{\dots, -2, -1, 0, 1, 2, \dots\}$ $-5 \in \mathbb{Z}$，$\frac{1}{2} \notin \mathbb{Z}$
$\mathbb{Q}$ 有理数集 所有能写成分数的数 $\frac{1}{2} \in \mathbb{Q}$，$\sqrt{2} \notin \mathbb{Q}$
$\mathbb{R}$ 实数集 有理数 $+$ 无理数 $\pi \in \mathbb{R}$
$\emptyset$ 空集 没有任何元素

🔑 关系：$\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}$（$\subset$ 表示"被包含于"，后面会讲）

五、集合之间的关系

子集：$\subseteq$

如果集合 $A$ 的每个元素都在集合 $B$ 中，那么 $A$ 是 $B$ 的子集，记作 $A \subseteq B$。

例如：$A = \{1, 2\}$，$B = \{1, 2, 3\}$，则 $A \subseteq B$（$B$ 包含了 $A$ 的所有元素）

一个容易混淆的点：$\emptyset$ 是任何集合的子集。因为"空集里的每个元素都在 $X$ 中"这个命题——由于空集里没有任何元素，所以这句话"平凡地"成立。

真子集：$\subset$

如果 $A \subseteq B$ 且 $A \neq B$（$B$ 中至少有一个元素不在 $A$ 中），则 $A$ 是 $B$ 的真子集，记作 $A \subset B$。

$\{1, 2\} \subset \{1, 2, 3\}$ ← 真子集（$\{1,2,3\}$ 有额外的元素 $3$）

$\{1, 2, 3\} \subseteq \{1, 2, 3\}$ ← 子集，但不是真子集（两者相等）

相等：$=$

两个集合 $A$ 和 $B$ 相等，当且仅当 $A \subseteq B$ 且 $B \subseteq A$。

换句话说，它们包含完全相同的元素。

六、集合的运算

交集：$\cap$

$A \cap B$ $= \{x \mid x \in A \text{ 且 } x \in B\}$

两个集合的公共部分。


A = \{1, 2, 3\}
B = \{2, 3, 4\}
A \cap B = \{2, 3\}

并集：$\cup$

$A \cup B$ $= \{x \mid x \in A \text{ 或 } x \in B\}$

两个集合加在一起（不重复算）。


A = \{1, 2, 3\}
B = \{2, 3, 4\}
A \cup B = \{1, 2, 3, 4\}

补集

在给定的"全集"中，不属于某个集合的元素组成的集合。


全集 U = \{1, 2, 3, 4, 5\}
A = \{1, 2\}
A 的补集 = \{3, 4, 5\}（在 U 中但不在 A 中的元素）

七、集合和函数的关系（预览）

后面我们会学到函数的概念。函数的本质就是从"一个集合"到"另一个集合"的对应规则。


定义域（x 的取值范围）→ 一个集合
值域（y 的取值范围）  → 另一个集合
函数                 → 连接两个集合的规则

现在你只需要记住：集合是一切的基础。

✅ 检查点

  1. 用两种方法表示"大于 $3$ 且小于 $8$ 的整数"组成的集合。
  2. 判断：$\emptyset \subseteq \{1, 2\}$？$\mathbb{N} \subseteq \mathbb{Z}$？$\mathbb{N} \subset \mathbb{Z}$？
  3. $A = \{1, 3, 5, 7\}$，$B = \{2, 3, 5, 8\}$，求 $A \cap B$ 和 $A \cup B$。
  4. $\mathbb{N}$、$\mathbb{Z}$、$\mathbb{Q}$、$\mathbb{R}$ 的关系是什么？用 $\subseteq$ 或 $\subset$ 写出它们之间的包含关系。
点击查看答案
  1. 列举法：$\{4, 5, 6, 7\}$；描述法：$\{x \in \mathbb{Z} \mid 3 < x < 8\}$
  2. 都对。$\emptyset \subseteq \{1, 2\}$（空集是任何集合的子集）；$\mathbb{N} \subseteq \mathbb{Z}$（自然数都是整数）；$\mathbb{N} \subset \mathbb{Z}$（$\mathbb{N}$ 是 $\mathbb{Z}$ 的真子集，$\mathbb{Z}$ 还有负数）
  3. $A \cap B = \{3, 5\}$；$A \cup B = \{1, 2, 3, 5, 7, 8\}$
  4. $\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}$
