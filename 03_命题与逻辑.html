03 命题与逻辑：思维的规则

数学不只是算数，更是一种精确的语言。本章教你怎样正确地"说对错"和"做推理"。

一、什么是命题？

📖 命题：一个可以判断真假的陈述句。

换句话说：一个句子，你看了之后能说"对"或者"错"，它就是命题。

句子是命题吗？真假？
句子是命题吗？真假？
"三角形的内角和是 $180^\circ$"是真
"$1 + 1 = 3$"是假
"今天天气真好"否— 无法判断，主观
"$x > 3$"否— 取决于 $x$，不确定
"请把门关上"否— 祈使句，没有真假

💡 判断标准：能不能说它是"对的"或"错的"？

⚠️ 注意：假命题也是命题！"$1 + 1 = 3$"虽然是错的，但它确实是一个可以判断真假的陈述句。

二、逻辑联结词：把命题串起来

1. 且（$\land$）—— "并且"

"$P$ 且 $Q$"为真 $\iff$ $P$ 为真同时 $Q$ 为真。

例："$2$ 是偶数 且 $2$ 是质数" → 真（两个都对） 例："$2$ 是偶数 且 $2$ 是奇数" → 假（第二个是错的）

通俗记忆：两个都对才算对，一个错就是错。

2. 或（$\lor$）—— "或者"

"$P$ 或 $Q$"为真 $\iff$ $P$ 为真或 $Q$ 为真（至少一个真）。

例："$2$ 是偶数 或 $2$ 是奇数" → 真（第一个对） 例："$3$ 是偶数 或 $5$ 是偶数" → 假（两个都错）

通俗记忆：有一个对就算对，全错才算错。

💡 数学中的"或"是包含性的——可以两个都成立。和日常语言中"你要茶或咖啡"（只能选一个）不同。

3. 非（$\lnot$）—— "不是"

"非 $P$"为真 $\iff$ $P$ 为假。

例："$2$ 是奇数"的否定："$2$ 不是奇数" → 真 例："三角形的内角和是 $180^\circ$"的否定 → "三角形的内角和不是 $180^\circ$" → 假

通俗记忆：反过来就行。

三、充分条件与必要条件

这是数学中最容易搞混的概念之一。用一个例子来记：

"如果下雨，那么地面湿。"
  • 下雨 $\Rightarrow$ 地面一定湿：下雨是"地面湿"的充分条件。
  • 地面湿 $\Leftarrow$ 可能下雨：地面湿是"下雨"的必要条件（地面如果不湿，肯定没下雨）。

🔑 口诀：

  • 充分条件 = "有这个就一定行"（$P$ 充分保证 $Q$）
  • 必要条件 = "没这个就一定不行"（没 $Q$ 就没 $P$）

更多例子：

命题充分条件必要条件
命题充分条件必要条件
"如果是正方形，那么是矩形"正方形矩形
"如果能被 $6$ 整除，那么能被 $3$ 整除"被 $6$ 整除被 $3$ 整除
"如果 $x > 5$，那么 $x > 3$"$x > 5$$x > 3$

📝 用集合来理解：


    必要条件（范围大）
    ┌─────────────────────┐
    │  充分条件（范围小）   │
    │  ┌─────────────┐    │
    │  │  正方形      │    │
    │  │             │    │
    │  └─────────────┘    │
    │      矩形            │
    └─────────────────────┘

小圈里的东西一定在大圈里（充分），但在大圈里不一定是小圈里的东西。

四、四种基本的命题变换

以"如果两个三角形全等，那么它们面积相等"为例：

变换形式例子真假
变换形式例子真假
原命题如果 $P$，则 $Q$如果全等，则面积相等真
逆命题如果 $Q$，则 $P$如果面积相等，则全等假！
否命题如果非 $P$，则非 $Q$如果不全等，则面积不等假！
逆否命题如果非 $Q$，则非 $P$如果面积不等，则不全等真

⚠️ 重要结论：

  • 原命题为真 不代表 逆命题为真！
  • 但 原命题 $\equiv$ 逆否命题（它俩要么都真、要么都假）

五、在数学证明中的运用

你学过的全等三角形判定，就大量使用了逻辑：

"如果两边及其夹角对应相等，那么两个三角形全等"（SAS 判定）

结构中：

  • 充分条件：两边及其夹角对应相等
  • 结论：两个三角形全等

当你使用 SAS 判定来证明两个三角形全等时，你的逻辑链是：

  1. 已知 $AB = DE$, $\angle B = \angle E$, $BC = EF$
  2. 根据 SAS 判定，（$AB=DE$ 且 $\angle B=\angle E$ 且 $BC=EF$）是（$\triangle ABC \cong \triangle DEF$）的充分条件
  3. 所以 $\triangle ABC \cong \triangle DEF$

这看起来可能很"显然"，但把逻辑结构说清楚，是学好数学的第一步。
